## 🔤 Tipografía

**Poppins** (Google Fonts) — geométrica, moderna, accesible.

- Pesos recomendados: **Medium (500)** y **SemiBold (600)** para el branding.
- Para UI: Regular (400) para body, Medium (500) para énfasis, SemiBold (600) para títulos.

Disponible gratis en: https://fonts.google.com/specimen/Poppins

## 💡 Concepto

La marca combina dos mundos:

- **La D geométrica robusta** = estructura, datos, solidez corporativa de un CRM serio.
- **El halo flotante** = lo celestial, la trascendencia, lo "divino" del nombre.
- **El nodo central** dentro de la D = conexión, integración con hardware, el corazón del sistema.
- **Los halos sobre las "i"** en el wordmark = detalle distintivo que liga lo tipográfico con el concepto.

## ✅ Buenas prácticas

- **Sí**: Usa siempre los archivos SVG provistos. Escalan sin pérdida.
- **Sí**: Mantén un área de seguridad alrededor del logo (mínimo = altura del halo).
- **Sí**: Sobre fondos oscuros → usa la versión blanca. Fondos claros → versión color o negra.

- **No**: No estires, deformes ni rotes el logo.
- **No**: No cambies los colores fuera de la paleta oficial.
- **No**: No coloques el logo sobre fondos visualmente ocupados sin un backdrop sólido.

---

## 🛠 Detalles técnicos

- Todos los SVG tienen el **texto convertido a paths** (curvas vectoriales). Esto significa que el logo se verá exactamente igual en cualquier dispositivo, sin depender de que la fuente Poppins esté instalada.
- Las versiones blancas usan `fill-rule="evenodd"` para tener huecos transparentes reales (no rellenos de color), lo que permite usarlas sobre cualquier fondo.
- Los gradientes están definidos con `linearGradient` estándar SVG — compatible con todos los navegadores modernos y software de diseño (Figma, Illustrator, etc.).
